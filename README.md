# 得点期待値をPythonで出すサンプル

セイバーメトリクス指標「得点期待値」をPythonで算出するサンプル. MLBのデータを使用.

## 推奨環境

- Python3.8
- poetryでパッケージ管理しているので事前に設定してもらえるといいかも
- 一応requirements.txtも準備しています

## 初期設定&インストール

### poetry

```bash
$ poetry install
```

### pip 

```bash
$ pip install -r requirements.txt
```

## 起動

### Jupyter

```bash
$ jupyter lab
```


### streamlit

```bash
$ streamlit run sample_run_expectancy.py
```
